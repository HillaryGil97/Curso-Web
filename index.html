Etiquetas Semanticas

header
Se reserva para el encabezado
nav
Se utiliza para los menus o tambien a agrupar, "es un navegador" puede tener otras funciones
section
Es para agrupar elementos o contenidos relacionados entre si.
article
Es para mantener organizados elementos que no necesariamente estan relacionados
aside
Publicidad o información extra que no necesariamente se relaciona con la página
footer
Se reserva para los derechos de autor
main
similar al section pero para agrupar la parte principal de la página
div
Sirve para seccionar pero tambien para dar formato con CSS
